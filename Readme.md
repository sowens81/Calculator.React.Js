# Calculator.React.Js

A React.JS Demo calculator application.

## Installation

Use the package manager [npm](https://nodejs.org/en/) part of node.js to install dependencies.

```bash
cd ./src/calculator.app/ 
npm install

```

## Usage

```bash
cd ./src/calculator.app/ 
npm start
```

## Run Unit Tests

```bash
cd ./src/calculator.app/ 
npm test
```
